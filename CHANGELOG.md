# Changelog

## [0.1.0] - 2026-03-31
### Toegevoegd
- Initieel framework met generieke AI bot class
- Claude Code integratie (non-interactive, --dangerously-skip-permissions)
- Zelflerende memory (conversations + knowledge base)
- Telegram formatter met HTML + plain text support
- Monitoring module (health checks, security scans, alerts)
- Plugin systeem (Proxmox, Home Assistant, MikroTik)
- YAML-based configuratie per bot instance
- Template systemd service voor meerdere bots
- Voorbeeldconfiguraties voor DevOps, Facility, en Executive bots
